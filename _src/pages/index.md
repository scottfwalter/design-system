## Development

1. Clone the [repo](https://github.com/peruvianidol/lwjds): `git@github.com:peruvianidol/lwjds.git`
2. Run `npm install`
3. Run `npm start` to start the [Eleventy](https://11ty.dev) server
4. Browse to `http://localhost:8080`
